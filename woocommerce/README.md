# WooCommerce Binance Pay Gateway

Accept USDT payments via Binance Pay in WooCommerce.

## Installation

1. Upload `wc-binance-pay/` to `/wp-content/plugins/`
2. Activate in **WordPress → Plugins**
3. Go to **WooCommerce → Settings → Payments**
4. Enable **Binance Pay (USDT)** and configure

## Publishing to WordPress.org

1. Create account at [wordpress.org](https://wordpress.org)
2. Submit plugin at [wordpress.org/plugins/developers/add](https://wordpress.org/plugins/developers/add)
3. Upload zip of `wc-binance-pay/` folder
4. Wait for review (usually 1-2 weeks)
